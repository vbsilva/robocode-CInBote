# robocode-Justice_League
University project aimed to learn the AI basics using robocode. Universidade Federal de Pernambuco - CIn - Brazil

No droolsrobocode.bat:
1. Robot -> Import robot or team

2. Selecione o arquivo justice_league-0.0.1-SNAPSHOT.jar na pasta exemplodrools\robocode-drools\target

3. Vá em robocodedrools\robocode\robots, Right click no arquivo importado e "Extract Here..." (Sobrescreva a pasta META-INF)

4. Copie a pasta exemplodrools\robocode-drools\reglas e todos os arquivos .java do diretório exemplodrools\robocode-drools\src\main\java\justice_league para dentro da pasta robocodedrools\robocode\robots\justice_league

5. Reinicie o Robocode

6. Battle -> New; Selecione o package justice_league e adicione dois robôs "RobotDrools*"; Start Battle
